# Backend часть проекта LangLearn

## О проекте

Данный проект является backend частью проекта LangLearn.
Здесь реализована регистрация, авторизация, работа с Cookie,
работа с базой данных.

## Технологии

* Node.js
* TypeScript
* Nest.js
* Mongodb
* Mongoose
* Cookie
* Git

___

[Ссылка для запрос к API](https://lang-learn-backend.vercel.app/)
___

## Установка

```bash
$ npm install
```

## Запуск приложения

```bash
# разработка
$ npm run start

# режим просмотра
$ npm run start:dev

# режим продакшена
$ npm run start:prod
```
